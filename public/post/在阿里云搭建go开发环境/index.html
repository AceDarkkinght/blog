在阿里云搭建go开发环境

  • Dec 11, 2017
  • Categories: golang , 阿里云
  • 1 min read

运行环境ECS Ubuntu 16.04 64位 #切换到安装文件夹 cd /usr/local #下载go #由于墙的原因，直接下载官方的可能会失败，这里用国内一个论坛的 wget https://dl.gocn.io/golang/1.8.4/go1.8.4.linux-amd64.tar.gz #解压 tar -zxvf go1.8.4.linux-amd64.tar.gz #创建工作目录 mkdir -p GOPATH goProject #设置环境变量，使用vim打开 vim /etc/profile export PATH=$PATH:/usr/local/go/bin #添加工作目录，可以添加多个工作目录，用:分隔 export GOPATH=/usr/local/GOPATH:/usr/local/goProject #退出后保存 source /etc/.profile 输入 go env如果看到go环境的信息说明配置成功 #安装Git sudo apt-get update sudo apt-get install git
